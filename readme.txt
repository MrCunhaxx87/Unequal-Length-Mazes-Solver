O Jogo recebe as partes dos códigos dos arquivos:    game.py, algorithms.py, interface.py ,maze.py e heuristics.py


1. Execute o arquivo main.py;
2. O jogo solicita que escolha qual o tipo de algortmo que gostaria de utilizar. escolha os valores 1, 2, 3, 4, 5 ou 6;
3. Será solicitado a escolha do Board Size. Escolha entre as opções 1, 2 ou 3
4. Após a escolha, os algoritmos fazem as pesquisas das melhores soluções.
5. Após a pesquisa encontrar a melhor solução, será executado a interface, mostrando o caminho escolhido.


OBS1: 
PARA INSTALAR AS DEPENDICIAS NECESSÁRIAS, BASTA DIGITAR O COMANDO NO TERMINAL      pip install -r requirements.txt

OBS2: 
É PRECISO INSTALAR A BIBLIOTECA PYGAME EM SUA MÁQUINA, CASO CONTRÁRIO A INTERFACE NÃO SERÁ EXECUTADA.
